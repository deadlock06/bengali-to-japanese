What changed in the rebrand

Display name → Bhasago across app_en/bn/ja.arb, main.dart, README. New lib/app/theme.dart (this palette) wired into MaterialApp. Package sensei_app, the SenseiApp class and sensei.db stay — renaming them would break the running build and existing user data.

Guardrails kept

Streak is a plain number (no loss warnings). Skip · Hint · Quit present on every learning screen. Bengali-first copy with optional EN/JA. Dark + calm for budget AMOLED.

Bhasago brand & UI kit · dark theme · generated preview. Fonts: Noto Sans Bengali / Noto Sans JP. Open in any browser — self-contained, no build step.
