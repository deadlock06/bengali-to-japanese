# AI agents: start in the `docs/` folder

This project's operating manual is the **`docs/` folder** — always use it first.

**Read order, every session:**
1. `docs/00_START_HERE.md` — router + the 7 non-negotiables; maps your task → the 1–2 spec files to load (don't read the whole spec).
2. `CODEBASE_MAP.md` (repo root) — what is built vs not built.
3. `NEXT_SESSION.md` (repo root) — last session's work + what's next.

Full details, hard rules, and verify commands: see **`CLAUDE.md`** (same directory). This file is a pointer for non-Claude agents; `CLAUDE.md` is the canonical entry.
